Core Engine

The high-performance, WebGL-accelerated rendering core for generating ultra-smooth, deeply layered liquid glass interfaces on the modern web.

npm install @pandahub/core-engine

Features

  • Sub-millisecond frosted glass blurring
  • Dynamic shadow depth calculation
  • Hardware-accelerated micro-interactions
  • Zero-config iOS 27 dark mode profiles
